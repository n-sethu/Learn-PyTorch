# Learning PyTorch

A structured, hands-on walkthrough of core PyTorch concepts, worked through module by module with documented exercises and implementations. The goal is to build genuine understanding of deep learning fundamentals rather than surface-level familiarity.

## Scope

The modules covered progress from tensor operations through full model construction:

- Tensors and tensor operations
- Automatic differentiation with autograd
- Training loops and optimization
- Neural network modules
- GPU usage and device management
- Data loading and preprocessing pipelines
- Custom layers and architectures

Each module has corresponding exercises and notes written in my own words.
